# Тема
## Цветовая тема
Выбираем цветовую тему по-умолчанию.
- Настройка > Color Theme
- Выбираем из списка подходящую

Устанавливаем цветовую тему через инструмент расширений:
- Extensions > Space Ocean Kit Refined

## Тема значков файлов
- Manage > File Icon Theme
